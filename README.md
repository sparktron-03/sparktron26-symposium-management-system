
# SPARKTRON'26 Symposium Management System

A Java-based symposium event management system developed for Thamirabarani Engineering College.

## Features
- Add Events
- View Events
- Delete Events
- Technical & Non-Technical Event Management

## Technologies Used
- Java
- ArrayList
- OOP Concepts
- Scanner Class

## Technical Events
- Paper Presentation
- Technical Quiz
- Circuit Debugging

## Non-Technical Events
- Anagram
- Play Quest
